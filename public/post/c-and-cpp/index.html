Por que aprender C e C++ antes de qualquer linguagem?

Por que aprender C e C++?

Antes de mais nada, é primeiramente importante enterdemos de onde surgiu, e como esta linguagem funciona. Bem, C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix.

Considerada uma linguagem de alto nível genérica, a C pode ser usada em diversos tipos de projeto, como a criação de aplicativos, sistemas operacionais, drivers, entre outros. Trata-se de uma linguagem estruturada que se tornou muito popular nos anos 80 — tanto que é difícil encontrar arquiteturas para as quais não existam compiladores para a C, o que garante o seu elevado nível de portabilidade.

Uma das maiores que você terá programando em C será a portabilidade, já que tanto C quanto C++ estamos presentes em praticamente todos os sistemas existentes, desde Windows, até o seu Android.

Além de outras:

  • Simplicidade
  • Confiabilidade
  • Velocidade
  • Poder

Ambas as linguagens são super poderosas, além de que são compiladas, ou seja, passa por um compilador, e após isso pode ser executada diretamente pelo sistema.
